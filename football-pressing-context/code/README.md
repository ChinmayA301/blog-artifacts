# Additional Ronaldo Context Scripts

These scripts extend the main `football-pressing-context` artifact with two
separate companion studies used by the Ronaldo/Portugal blog. They are not the
same data as the StatsBomb pressing study generated by `../generate_figures.py`.

## Files

| File | Purpose |
|---|---|
| `physical_output_context_study.py` | Recreates the aggregate physical-output comparison used to keep "Ronaldo does not move" claims in context. |
| `screenshot_trap_study_template.py` | Creates a transparent manual-coding template for testing viral screenshot claims against repeated comparable situations. |

## Running

From the repository root:

```bash
python football-pressing-context/code/physical_output_context_study.py
python football-pressing-context/code/screenshot_trap_study_template.py
```

`physical_output_context_study.py` runs from hard-coded public aggregate rows.
`screenshot_trap_study_template.py` creates a blank coding template and demo
summary output; replace the demo rows in `code/outputs/screenshot_trap_coded_events.csv`
with manually coded events for a real screenshot-trap audit.

Both scripts write generated outputs to `football-pressing-context/code/outputs/`,
which is ignored by git.
